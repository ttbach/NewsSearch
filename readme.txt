To compile:
	1) cd to NewsSearch
	2) Enter the following commands:
		a) $ chmod +x gradlew
		b)$ ./gradlew assembleDebug
	The api file is then stored in NewsSearch/app/build/outputs/apk (already present there)

To run the apk file:

     Using a phone: 
	-Install the app into the android phone using an install manager or by copying it into a sd card and then installing it.

     Using an emulator from AndroidStudio (can be downloaded from: http://developer.android.com/sdk/index.html): 
		
        - Import the project
	- Start the emulator (go to AVD manager to start the emulator)		- run the app (run option available in the studio).
